1. Basic Column Grid

Bootstrap grids use a column system. You can divide your layout into columns using classes like col-4.

col-4

2. Responsive Grid

You can make grids responsive by using breakpoint-specific column classes like col-sm-6 or col-md-4 to change layouts across screen sizes.

col-sm-6 col-md-4
col-sm-6 col-md-4
col-sm-12 col-md-4
<div class="row"> <div class="col-sm-6 col-md-4">col-sm-6 col-md-4</div> <div class="col-sm-6 col-md-4">col-sm-6 col-md-4</div> <div class="col-sm-12 col-md-4">col-sm-12 col-md-4</div> </div>

3. Nested Grid

You can nest rows and columns inside other columns to build more complex layouts.

Nested col-6
Nested col-6

4. Column Offsets

Offsets let you shift columns to the right using classes like offset-md-2.

offset-md-2
col-md-4
<div class="row"> <div class="col-md-4 offset-md-2">offset-md-2</div> <div class="col-md-4">col-md-4</div> </div>

5. Column Ordering

Use order-* classes to rearrange the visual order of columns without changing the HTML structure.

First Visually
Third Visually
Second Visually
<div class="row"> <div class="col order-3">First Visually</div> <div class="col order-1">Third Visually</div> <div class="col order-2">Second Visually</div> </div>

To remove space (gutters) between columns, use the g-0 class on the row.
