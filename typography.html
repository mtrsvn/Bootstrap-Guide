2. Lead Text

This is an example of lead text. It’s larger and slightly bolder than regular text. 

<p class="lead">
  This is an example of lead text. It’s larger and slightly bolder than regular text.
</p>

3. Blockquote

This is a blockquote example. It’s used to display large quoted text, often with some spacing and a border.


Someone famous in Source Title

  <p class="mb-0">This is a blockquote example. It’s used to display large quoted text, often with some spacing and a border.</p>

4. Inline Text Elements

This is highlighted text using the mark element. And this is deleted text using the del element. This is inserted text using the ins element. This is underlined text using the u element. 

<p>
  This is <mark>highlighted text</mark> using the <code>mark</code> element. And this is <del>deleted text</del> using the <code>del</code> element. This is <ins>inserted text</ins> using the <code>ins</code> element. This is <u>underlined text</u> using the <code>u</code> element.
</p>
